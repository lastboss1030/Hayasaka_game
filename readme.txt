・爆発アニメーションや弾発射時のエフェクト
・テキストでの敵配置

【シューティング作品説明】
C++で制作した縦スクロールシューティングゲームです。
基本的にはボス周辺にこだわり、制作しました。
最初は一つの大きなパーツだけだったのですが、
パーツ分けをすることによって、ボスからの攻撃方法や
プレイヤーの操作がパターン化されないようにしました。
ボス前の敵の配置はテキストファイルで管理を行えるようにしました。

他にもエフェクトや爆発アニメーションも、出来るだけきれいに、
いかにそれっぽく見せるかどうかで試行錯誤しました。

【パーティクルツール】
C++で制作したパーティクルツールです。
自分でパラメータを変更しながら表示することが出来ます。
パラメータをテキストファイルで保存、読み込みを出来るようにしました。

【敵生成ツール】
C++で制作した敵生成ツールです。
テキストファイルで座標やタイプ、ライフなどを読み込むことが出来ます。
このツールで配置等を確認し、そのままゲーム作品に使うことが可能です。
